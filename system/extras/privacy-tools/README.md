# Privacy Tools

Command-line utilities for privacy verification and system hardening.

## Tools

### privacy-check
Scans system for Google services and tracking components.

```bash
adb shell privacy-check
```

### network-monitor
Real-time network connection monitoring.

```bash
adb shell network-monitor
```

### tracker-scan
Analyzes installed apps for known trackers.

```bash
adb shell tracker-scan [package-name]
```

## Installation
Built and installed automatically with AndyOS.
