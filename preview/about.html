Key Features

Per-app network firewall and tracker blocking. Monitor and control all network activity.

Hardened Security

Open-source app ecosystem. No account required, no tracking.

Privacy DNS

OTA Updates

Secure over-the-air updates. Stay current without compromising privacy.

Frequently Asked Questions

Common questions about AndyOS

Is AndyOS compatible with my device?

AndyOS currently supports Google Pixel devices (Pixel 7 and newer) and x86_64 virtual machines (VirtualBox, VMware). More device support is planned.

Will my banking apps work?

Most apps work fine, but some banking apps may require SafetyNet/Play Integrity. We're working on compatibility solutions while maintaining privacy.

Can I install Google apps if needed?

Yes, you can install microG or sandboxed Google Play Services if absolutely necessary, though we recommend privacy-focused alternatives.

How do I get updates?

AndyOS includes a built-in OTA updater that securely downloads and installs updates from our servers without any tracking.

Is AndyOS free?

Yes! AndyOS is completely free and open source under the Apache 2.0 license. No ads, no subscriptions, no hidden costs.

How can I contribute?

Check our Contributing guide on GitHub. We welcome code contributions, bug reports, documentation improvements, and translations.
