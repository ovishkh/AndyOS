# AndyOS Browser (Bromite Fork)

Privacy-hardened browser based on Bromite/Vanadium.

## Features
- Ad and tracker blocking built-in
- No Google services integration
- Enhanced privacy controls
- Hardened security settings
- DNS over HTTPS support

## Configuration
- Default search: DuckDuckGo
- DNS: Quad9
- No telemetry
- No crash reporting
