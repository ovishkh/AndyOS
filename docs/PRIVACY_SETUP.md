# AndyOS Privacy Configuration

## Essential Settings

### 1. DNS over HTTPS
Settings > Network > Private DNS. Set: `dns.quad9.net`.

### 2. Network Privacy
AndyOS Settings > Network Privacy. Enable: "Block all trackers".

### 3. App Permissions
AndyOS Settings > App Permissions. Set: "Deny by default".

### 4. Location Privacy
Settings > Location. Mode: "Device only" (no network).

### 5. Firewall Rules
AndyOS Privacy Guard > Firewall. Deny background data for all apps except essential ones (e.g., Signal, K-9 Mail).

## Advanced Privacy

### VPN Always-On
Settings > Network > VPN.
- Enable: Always-on VPN.
- Enable: Block connections without VPN.

### MAC Randomization
Settings > Network > WiFi. Advanced > Privacy: Use randomized MAC.
